Although I am not very familiar with Python, I'll try to take a crack at this question.
The difference is semantic. 

Class attributes are attributes which  are owned by the class itself. As such, they  are shared by all the instances of the class.  Therefore, they have the same value for every instance.
And of course, when you change the value of a class attribute, if will affect all instances, since they share the exact same value.

On the other side, instance attributes belong to the instance and are therefore unique to that instance.

You can take a look at the link below, which discusses briefly the topic with code examples :
9. Classes - Python 2.7.10 documentation
